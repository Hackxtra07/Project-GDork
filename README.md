# Project-GDork 🔍

Advanced Google Dork Generator Pro is a powerful Python-based tool designed for security researchers, penetration testers, and OSINT analysts to generate high-precision Google Dorks.

## ✨ Features

- **Advanced Dork Engine**: Generates thousands of variations using categorized patterns (Vulnerabilities, IoT, Credentials, etc.).
- **Hyper-Specific Generation**: Automatically injects search operators (inurl, intitle, intext) into user keywords for surgical precision.
- **Bulk Domain Support**: Generate dorks for multiple domains simultaneously.
- **Rich UI**: Built with Tkinter, featuring a dark mode aesthetic and status monitoring.
- **Export Options**: Save your results directly to TXT, CSV, or JSON.
- **Smart Filtering**: Built-in safety filters to avoid common junk results.

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- Tkinter (usually comes with Python)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Project-GDork.git
   cd Project-GDork
   ```

2. Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Usage

Run the main script:
```bash
python main.py
```
